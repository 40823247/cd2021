Others

Fix cmsimde

1.進入https://github.com/mdecourse/cmsimde

2.點commits

3.找到最新版本，點旁邊那串

4.把網址後面那串複製下來

5.進小黑窗，到cmsimde，輸入git checkout 把剛剛那串貼上，按ENTER


6.回到cd2021，PUSH一次

Update cmsimde

1.打開黑窗，進入cmsimde

2.輸入git pull origin master，按enter

3.打開資料夾>cd2021>cmsimde>up_dir

4.複製acp.bat、cms.bat、http-server.py、localhost.crt、localhost.key

5.回到cd2021，貼上，系統會問你要不要取代http-server.py，按取代檔案

用完之後，以後想要打開遠端，只需要進到cd2021，輸入cms即可。

要推送的話，則是進到cd2021，輸入acp "上傳標題"即可。
